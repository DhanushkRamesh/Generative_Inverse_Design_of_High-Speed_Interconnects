Mathematical Framework of Vector Fitting:

The goal is to approximate frequency response (fs) as a rational function: ![[Pasted image 20260310213621.png]]

This is inherently a mathematical problem because the pole (an) in the denominator is unknown. Vector fitting bypasses this by solving the problem sequentially as two linear stages. 

Stage 1: Pole relocation:
1. Starting Poles: User specifies a set of initial staring poles
2. The unknown function: The function (fs) is multiplied by the unknown scaling function σ(s) = ∑n=1N​s−aˉn​c~n​​+1
3. The Linear Problem: This creates and augmented problem ∑s−aˉn​cn​​+d+sh≈(∑s−aˉn​c~n​​+1)f(s), because the starting poles (an) is known and the equation becomes linear.
4. Least squares: By writing this writing this equation for several frequency points, this becomes overdetermined linear matrix problem which is solved via standard least squares to solve the unknown.
5. Finding the new poles: The new poles of the system are calculated at the zeros of fitted  σ(s) functions.
6. Enforcing Stability: If any newly calculated poles are unstable, the algorithm simply inverts the sign of their real parts. forcing them into a stable left half-plane before the next step.

Stage 2: Residue Identification: Once the true poles are found in stage 1, they are plugged back into the original approximation equation. Since the poles are now known constants, the equation is linear again. It is solved one fi0nal time to find the exact residues(cn), proportional constant (d), derivative term (h). If the fits are not perfect, the calculated poles can be used as the new starting poles for the another iteration.

The authors provide strict mathematical guidelines to avoid poor conditioning of the linear problem in stage 1

For functions with resonance peaks:  The Starting poles must be complex conjugate pairs distributed linearly over the frequency range. To avoid ill-conditioning, the real part should be set exactly to 1% of the imaginary part.
