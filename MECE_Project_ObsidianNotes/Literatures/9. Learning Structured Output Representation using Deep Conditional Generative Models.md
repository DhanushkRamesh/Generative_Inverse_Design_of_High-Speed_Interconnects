In this paper the authors addresses the limitation of using standard convolution neural networks for structured output prediction. Standard CNN are deterministic - meaning they are used to model a  many-to-one function mapping. but in a structured prediction, a single input might have a multiple valid output (one-to-many mapping.) When a deterministic CNN are forced to model this - they result in an averaged unrealistic possibilities. 

To solve this, the authors propose Deep Conditional Generative Model (CGM) that used stochastic gaussian latent variables to model the diverse conditional distribution of output variables.

They build the model upon the framework of standard Variational Autoencoders (VAE).

The authors also notice a critical flaw in the CVAE training: the discripency between the training pipeline and the testing pipeline. 

During training the model uses the recognition network qœï‚Äã(z‚à£x,y) to draw z. It sees the true output y, making the reconstruction relatively easy. 

During Testing: the true output y is unknown, so the mode must rely purely on the prior network pŒ∏‚Äã(z‚à£x) to draw z.

Because the model was trained using the easier path, its performance degrades during testing. To fix this, the authors introduce Gaussian Stochastic Neural Networks (GSNN)

- Concept of GSNN: The GSNN forces the training pipeline to match the testing pipeline by setting the recognition network equal to the prior network. qœï‚Äã(z‚à£x,y)=pŒ∏‚Äã(z‚à£x)
- This eliminates the KL divergence term completely, simplifying the objective to purely reconstructing term.

The authors further improve the performance by introducing a robust prediction strategy- two training strategies:

1. Multi-Scale Prediction: Instead of predicting the output as single resolution, the network is trained to predict output at multiple scales. This forces the model to learn the global-to-local, coarse-to-fine-grained prediction. 
2. Input Omission Noise: During training they randomly generate square masks and set those input pixels to 0.  This simulates occlusion and forces the network to learn to predict the missing information based on stochastic latent variables.
