This paper addresses the core problem in deep neural networks - which are notoriously vulnerable to input perturbations, meaning, a tinny change in the input data (like a random noise) can alter the model's prediction. 

A well engineered system needs a classification margin - a buffer zone around each data point where the model's decision remain stable. 

The authors propose Jacobian Regularization, a technique which explicitly forces the network to enlarge its decision cells and push boundaries away form the training points, hence increasing robustness.

Innovation - effective approximate algorithm

While the Jacobian regularization math is sound, computing the exact Jacobian norm in deep learning is computationally heavy.
